# 📘 SEN306 – Software Construction

This repository contains materials, projects, and exercises for the **SEN306: Software Construction** course. The focus of this course is on the principles, tools, and practices involved in building reliable, maintainable, and efficient software systems.

---

## 📌 Course Overview

Software Construction covers the practical aspects of software development, including:

- Writing clean and maintainable code  
- Applying software design principles  
- Understanding programming paradigms  
- Testing and debugging techniques  
- Using modern development tools and workflows  

---

## 🎯 Learning Objectives

By the end of this course, you should be able to:

- Apply best practices in coding and software design  
- Develop modular and reusable components  
- Use version control effectively (Git & GitHub)  
- Write and execute unit tests  
- Debug and optimize software systems  
- Collaborate efficiently in a team environment
