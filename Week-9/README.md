# SEN306: Week 9 - ADT 

**Author:** Udochukwu Ezinna  


This repository contains the practical implementations for the SEN306 session on Abstract Data Types, Information Hiding, and Inheritance.





## How to Run
Compile and run the `Main.java` file from the command line:
`javac *.java`
`java Main`

## Contents
* **Queue ADT**: Explores information hiding by implementing a `QueueADT` interface using both an `ArrayList` and a custom singly `LinkedQueue`.
* **Bank Account**: Demonstrates the fragility of inheritance using a `BankAccount` base class and an `OverdraftAccount` subclass.
